What are the differences between presenting an image using an HTML tag (like <img>) and presenting it as the background of an HTML element (using the CSS property background-image)?

HTML tag: Used when the image is related to the content and holds significant information.

Background: Only used when the purpose is to enhance the aesthetics of the webpage style.

Besides background-image, what other CSS properties affect the display of background images?

  • Background-attachment: Determines whether a background image is fixed in that position or if it moves when the page scrolls.
  • Background-clip: Determines how far the background extends; border-box("extends behind the border"), padding-box("extends to the inside edge of the border"), and content-box("extends to the edge of the content box").

    Dimensions is the actual size of the image (width and height), while Resolution is the density of pixels in the image; higher resolution/sharper image uses more pixel density.
