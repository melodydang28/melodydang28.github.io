Code Log 1: HTML Semantic Tags

What is the difference between non-semantic and semantic tags in HTML?

Non-semantic tags don't clearly define their purpose or meaning in the name itsself like semantic tags do, making them more generic. Semantic tags describe the kind of content they will contain and provides more information for better understanding.

What are two (or more) benefits of using semantic tags?

Search Engine Optimisation: Semantic tags impact a webpage's rankings in Google's algorithms by its placement of certain semantic keywords they use.

Semantic tags make content on a webpage more accessible through better web navigation, benefiting people with disabilities to navigate them easily.

Semantic tags make code more readable to others with the descriptive semantic names that already states its main purpose.
