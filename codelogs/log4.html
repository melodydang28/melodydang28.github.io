What do these CSS properties do?

  • Color: Represents color in their specified sections/classes through different color keywords, hex codes, RGB etc.
  • Background-color: Sets the background color of the section by filling up its entire area.
  • Border-color: Sets the color of the element's border. The border-color property can also specify individual borders and assign colors to them seperately.

How does CSS accept color as value for the following?

  • Named: color: blue;
  • Hexidecimal: color: #FF0000;
  • RGB: color: rgb(0%, 100%, 0%);
  • HSL: color: hsl(120,100%,50%);
