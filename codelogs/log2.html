Code Log 2: CSS Measuring Units

The difference between absolute and relative CSS length units:

Absolute units: Units that are not relative to anything, always the same size.

Relative units: Units that are relative to other measurements.

Unit descriptions:

px: Pixel

pt: Point

vw, vh: 1% of the width/height of the viewable screen.

Q: ¼ of a millimeter

cm: One centimeter

mm: One millimeter
